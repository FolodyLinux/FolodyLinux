# FolodyOS
FolodyOS is a simple, semi-rolling Arch-based Linux distro made by Folody Community.
![img](https://i.imgur.com/LRxYXoc.png)

# Installation
- Download FolodyOS iso
- Flash iso into an usb
- Boot into usb
- Run `Install System`
